# Germiani IA Studio 2.5 Flexh

Plataforma para gerenciar projetos, enviar arquivos para o GitHub, executar GitHub Actions e gerar versões para Windows e Android.

## Funcionalidades

- Envio de projetos completos para o GitHub
- Execução automática do GitHub Actions
- Acompanhamento de compilações em tempo real
- Download de arquivos .exe e .apk
- Histórico de projetos e execuções
- Configuração de chaves e tokens

## Instalação

```bash
npm install
npm run dev
```

## Deploy na Vercel

1. Faça push deste repositório para o GitHub
2. Acesse vercel.com e importe o repositório
3. A Vercel detecta automaticamente o Vite e faz o deploy

## Configuração

1. Abra o app
2. Vá na aba **Chaves**
3. Informe seu GitHub Token, usuário e repositório
4. Clique em **Testar Conexão** para validar
5. Vá em **Enviar Projeto** para fazer upload
